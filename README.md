# Tic-Tac-Toe

A simple Tic-Tac-Toe game developed in **Python**. The game is played against an AI opponent and focuses on the core gameplay without additional features or menus.

## Features

- Single Player (Player vs. AI)
- All winning conditions implemented
- Automatic win and draw detection
- Simple and lightweight user interface
- Bitboards are used for faster board calculations

## Requirements

- Python 3.x
- PyGame
